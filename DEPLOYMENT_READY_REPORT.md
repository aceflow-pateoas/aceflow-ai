### 部署步骤

#### 1. 环境准备
```bash
# 安装统一服务器
uvx aceflow-mcp-server@latest

# 验证安装
aceflow-unified --version
aceflow-unified --health-check
```

#### 2. 配置迁移
```bash
# 自动迁移现有配置
aceflow-unified --auto-migrate

# 验证迁移结果
aceflow-unified --migration-status
aceflow-unified --validate-config
```

#### 3. 功能验证
```bash

```
